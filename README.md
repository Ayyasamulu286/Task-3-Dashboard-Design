# Task 3 - Dashboard Design

## 📌 Objective

Design an interactive business dashboard using Power BI to help stakeholders analyze sales performance, profitability, customer behavior, and business trends for better decision-making.

---

## 🛠 Tools Used

* Power BI
* Microsoft Excel / CSV Dataset
* GitHub

---

## 📊 Dataset

**Sample Superstore Dataset**

The dataset contains information about:

* Orders
* Sales
* Profit
* Customers
* Products
* Categories
* Regions

---

## 🎯 Dashboard Features

### KPI Cards

* Total Sales
* Total Profit
* Total Orders
* Total Customers

### Visualizations

* Category Wise Sales Analysis
* Region Wise Sales Distribution
* Monthly Sales Trend Analysis
* Profit by Sub-Category
* Top 10 Customers
* Loss Making Products

### Interactive Elements

* Region Slicer
* Category Slicer
* Date Slicer

---

## 📈 Business Insights

* Technology category generated the highest sales.
* West region contributed the highest revenue.
* A few products generated losses despite having sales.
* Sales showed significant growth during year-end months.
* Top customers contributed a major share of total revenue.
